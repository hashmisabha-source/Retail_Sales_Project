# 🛒 Retail Sales Analysis Project

## 📌 Project Overview

This project analyzes retail sales data using SQL, Python, Pandas, NumPy, Excel, and Power BI. The objective is to clean the data, perform business analysis, and build an interactive dashboard for decision-making.

---

## 🎯 Project Objectives

- Clean the dataset
- Remove duplicate values
- Handle missing values
- Perform Exploratory Data Analysis (EDA)
- Analyze sales and profit trends
- Build an interactive Power BI dashboard
- Generate business insights

---

## 🛠️ Tools & Technologies

- SQL (PostgreSQL)
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Excel
- Power BI

---

## 📊 Key Performance Indicators (KPIs)

- Total Sales: **36M**
- Total Profit: **7M**
- Total Orders: **500**
- Average Sales: **73K**
- Average Profit: **14K**
- Total Quantity Sold: **2K**

---

## 📈 Business Insights

- Technology category generated the highest sales.
- Mumbai recorded the highest profit among all cities.
- Monthly sales trends helped identify peak-performing months.
- Power BI dashboard enables interactive filtering using slicers.
